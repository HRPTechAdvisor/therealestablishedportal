Adding an overlay to OBS (Open Broadcaster Software) is pretty straightforward. Here’s how to do it through the Sources panel:

🔹 Step-by-Step: Add an Overlay in OBS

  1. Open OBS Studio.

  2. Select the Scene you want to add the overlay to (or create a new scene if you haven’t yet).

  3. In the Sources panel (bottom of the screen), click the ➕ plus (+) button to add a new source.

  4. Choose one of the following options based on your overlay file:

    • 🖼️ Image – if your overlay is a static image (PNG, JPEG, etc.):

      • Name it (e.g., “Stream Overlay”).

      • Click Browse and select your overlay image.

      • Click OK.

    • 🎞️ Media Source – if your overlay is an animated file (like a .webm or .mp4 with transparency):

      • Name it and click OK.

      • Browse and select your overlay.

      • Check Loop if you want it to repeat.

      • Check Use hardware decoding if your PC supports it for better performance.

      • Check Restart playback when source becomes active if needed.

      • Click OK.

    • 🧑‍💻 Browser – if your overlay is a URL (from services like Streamlabs, StreamElements, etc.):

      • Name it and click OK.

      • Paste the overlay URL.

      • Set the resolution (usually 1920x1080 or whatever your canvas is).

      • Click OK.

  5. Resize and position the overlay on the preview canvas as needed. Use the red bounding box to adjust.

  6. Click the Lock icon 🔒 next to the source to prevent accidentally moving it during streaming.

✅ Pro Tips

  • Use PNG for overlays if you want transparent backgrounds.

  • Keep your overlays organized by naming them properly.

  • Use “Group” sources if you want to move or control multiple overlays together.
